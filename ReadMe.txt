README

- Les fichiers db_enzyme.txt et intenz.txt sont les fichiers dentree
- Les fichiers enzymeInsertion.txt et publiInsertion.txt sont des fichiers de requetes qui permettent de remplir la base de donnee
- Le ficher functions.php contient des fonctions pour manipuler la base de donnee

pour remplir la base : lancer le fichier bd.php
pour afficher les informations sur lenzyme EC 1.1.1.1 : lancer le fichier requetes.php

Pour afficher la liste de tous les enzymes de la bases -> lancer le fichier list.php.
